# VaMi Bags Mini App

Это веб-приложение для Telegram Mini App, которое позволяет пользователям оформлять заказы на сумки и подстаканники.

## Структура проекта

```
webapp/
├── index.html          # Основной HTML файл
├── styles.css          # Стили приложения
├── app.js             # JavaScript логика
├── manifest.json      # Манифест Mini App
├── server.py          # Веб-сервер
└── images/            # Директория с изображениями
    ├── shapes/        # Изображения форм
    └── materials/     # Изображения материалов
```

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.7 или выше
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите веб-сервер:
   ```bash
   python server.py
   ```
4. Приложение будет доступно по адресу: http://localhost:8000

## Настройка в Telegram

1. Создайте нового бота через @BotFather
2. Включите Mini App для вашего бота:
   ```
   /mybots
   Выберите вашего бота
   Bot Settings > Menu Button > Configure Menu Button URL
   ```
3. Укажите URL вашего Mini App (например, https://your-domain.com)
4. Сохраните изменения

## Разработка

- Для локальной разработки используйте `server.py`
- Для продакшена рекомендуется использовать nginx или Apache
- Убедитесь, что все изображения оптимизированы для веба
- Тестируйте приложение на разных устройствах

## Безопасность

- Всегда используйте HTTPS в продакшене
- Проверяйте данные, полученные от пользователя
- Храните конфиденциальные данные в безопасном месте
- Регулярно обновляйте зависимости

## Поддержка

При возникновении проблем:
1. Проверьте логи сервера
2. Убедитесь, что все файлы на месте
3. Проверьте права доступа к файлам
4. Проверьте настройки CORS
